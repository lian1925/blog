micro:bit 入门系列

作者： Lian

日期： 2017年11月03日 16:11

一、micro:bit 软件系统

二、PXT for micro:bit

开发环境安装

2.1 安装 pxt 命令

npm install -g pxt

2.2 下载 micro:bit 项目并安装项目依赖

git clone https://github.com/microsoft/pxt-microbit
cd pxt-microbit
npm install

2.3 运行项目

pxt serve

等待数秒，项目运行完成将自动打开浏览器展示效果

注意：若是需要使项目中的c++文件更改生效，则使用如下命令

pxt serve -yt

2.4 更新PXT版本

pxt update

2.5 测试

  • 测试是否正确生成.hex文件（最终运行于micro:bit硬件的编译文件）
    cd ./test  // 进入test目录
pxt testdir //此命令将对示例脚本进行编译，生成.hex文件

  • 单元测试，针对单一功能测试

    cd ./libs/lang-test*
pxt run

    2.6 生成目标文件,执行如下代码，则在built/packaged目录生成目标文件

    pxt staticpkg
