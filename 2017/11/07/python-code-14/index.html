课14 列表与循环 | Python

作者： Lian

日期： 2017年11月07日 18:11

我们已经知道，使用条件判断，可以让我们的代码变得聪明，做出一些智能的判断。然而，我们的程序还需要具备完成重复任务的能力。列表与循环的知识将赋予程序更强的威力。

新知识

我们看看如何创建列表:
hairs = [‘brown’, ‘blond’, ‘red’]
eyes = [‘brown’, ‘blue’, ‘green’]
weights = [1, 2, 3, 4]

代码练习

the_count = [1, 2, 3, 4, 5]
fruits = ['apples', 'oranges', 'pears', 'apricots']
change = [1, 'pennies', 2, 'dimes', 3, 'quarters']

# this first kind of for-loop goes through a list
for number in the_count:
    print "This is count %d" % number
# same as above
for fruit in fruits:
    print "A fruit of type: %s" % fruit

# also we can go through mixed lists too
# notice we have to use %r since we don't know what's in it
for i in change:
    print "I got %r" % i

 # we can also build lists, first start with an empty one
 elements = []

 # then use the range function to do 0 to 5 counts
for i in range(0, 6):
    print "Adding %d to the list." % i
    elements.append(i)

 # now we can print them out too
for i in elements:
    print "Element was: %d" % i

常见问题

1、为什么 for i in range(1, 3): 只循环 2 次而非 3 次?
range()函数会从第一个数到最后一个,但不包含最后一个数字。所以它在 2的时候就停止了,而不会数到 3。这种含首不含尾的方式是循环中及其常见的一种用法。
2、elements.append() 是什么功能?
它的功能是在列表的尾部追加元素。打开 Python 命令行,创建几个列表试验一下。以后每次碰到自己不明白的东西,你都可以在 Python 的交互式命令行中实验一下。
