课3 异步与文件读取

作者： Lian

日期： 2017年11月08日 16:11

本节将学习使用Node.js读取文件的操作。Node.js提供了两种文件读取方式：同步与异步。

知识点

关于读取文件，我们将学习一下的新语句。

读取
异步读取文件：readFileSync()
同步读取文件：readFile()

写入
异步写文件：writeFileSync()
同步写文件：writeFile()

代码练习

本节我们需要准备两个文件，一个是正常的txt文件，命名为lesson3_data.txt，写入如下内容：

Hello Node.js.

另一个文件命名为lesson3.js，填写如下代码：

const fs = require('fs');
const data = fs.readFileSync('/lesson3_data.txt'); // 同步操作，会造成堵塞
console.log("同步："+data);

// 异步读取文件
fs.readFile('/lesson3_data.txt', (err, data) => {
  if (err) throw err;
  console.log("异步："+data);
});

fs.writeFileSync('/lesson3_data.txt', '\n add something Sync.');

fs.writeFile('/lesson3_data.txt', 'Hello Node.js','w', (err) => {
  if (err) throw err;
  console.log('The file has been saved!');
});

OK，代码编写完，打开命令行终端，输入内容：
node app.js

常见问题

1、有哪些修饰符可以用来控制文件访问?

答：最重要的是 + 修饰符,写法就是 ‘w+’, ‘r+’, ‘a+’ ——这样的话文件将以同时读写的方式打开,而对于文件位置的使用也有些不同。
