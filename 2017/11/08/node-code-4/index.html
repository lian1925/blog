课4 express 入门

作者： Lian

日期： 2017年11月08日 17:11

Express 是一个基于 Node.js 平台web 应用框架，也是当前最受欢迎的开发框架。

本节将学习如何使用express构建基本的网站。

代码练习

1、创建一个目录lesson4,并进入目录

mkdir lesson4
cd lesson4

2、创建package.json文件，命令如下：

npm init

3、安装express，命令如下：

npm install express --save

打开atom，新建文件，命名为app.js，放入lesson4目录，编写代码如下：

var express = require('express');
var app = express();

app.get('/', function (req, res) {
  res.send('Hello World!');
});

var server = app.listen(3000, function () {
  var host = server.address().address;
  var port = server.address().port;

  console.log('Example app listening at http://%s:%s', host, port);
});

OK，代码编写完，打开命令行终端，输入内容：
node app.js

你应该看到的结果

打开浏览器，访问链接http://localhost:3000。
如果你看到页面返回”Hello World”，恭喜你，操作成功。

(完)
