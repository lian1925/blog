博客添加代码高亮特性

作者： Lian

日期： 2017年10月05日 05:10

目标：为博客添加代码高亮特性，让代码部分的观赏性加分。

工具

hexo-prism-plugin

安装

npm i -S hexo-prism-plugin

配置

在博客的配置文件_config.yml，加入配置语句：

prism_plugin:
  mode: 'preprocess'    # realtime/preprocess
  theme: 'duotone-light'
  line_number: false    # default false

禁止默认 highlight配置：

highlight:
  enable: false

修改样式

将代码块设为圆角，添加浅浅边框，修改后视觉效果更加柔和美观。
这里我们只需要修改pre选项，因为所有的代码块都包含在pre节点内,为避免干扰其他使用pre的样式，可加.page前缀。

.page pre {
  border: 1px solid rgba(0, 0, 0, 0.3);
  border-radius: 10px;
}

清除缓存并启动

hexo clean
hexo s
