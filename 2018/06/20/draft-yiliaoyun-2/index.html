作者： Lian

日期： 2018年06月20日 14:06

一 json web token

1、安装token库

npm install --save jsonwebtoken

2、生成token

var jwt = require('jsonwebtoken');

jwt.sign({
  exp: Math.floor(Date.now() / 1000) + (60 * 60),
  data: 'foobar'
}, 'secret');

3、验证token

// verify a token symmetric
jwt.verify(token, 'secret', function(err, decoded) {
  console.log(decoded.foo) // bar
});

二 登录接口API

1、登录，获取token

1、网关：/user/login post

访问格式： ip + 端口 + 网关
如： http://172.20.10.3:3000/user/login

2、请求参数：
{
username:’admin’,
password:’123’
}

3、返回：
// 正确
{
err_code: 0,
token: ‘xxx’
}

// 密码错误
{
err_code:1,
token: null
}

2、获取用户信息

1、网关： /user/info post

2、请求参数：
{
token: ‘xxx’
}

3、返回：
// 正确
{
err_code: 0,
username: ‘admin’,
age: 20
}

// 错误
{
err_code:1,
username:null
age:null
}
