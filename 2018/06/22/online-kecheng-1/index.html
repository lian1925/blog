1 Sass Loader

一种webpack加载器，用于加载 sass/scss 文件，并将其编译成css

1、安装

npm install sass-loader node-sass webpack --save-dev

sass-loader需要node-sass 和 webpack 作为同辈依赖(peerDependency)。
通俗来讲，如果你要安装sass-loader，你最好同时安装node-sass 和 webpack。

2、示例
一般情况下，安装css-loader和style-loader，以更好兼容常见样式。

npm install style-loader css-loader
--save-dev

配置

// webpack.config.js
module.exports = {
    ...
    module: {
        rules: [{
            test: /\.scss$/,
            use: [
                "style-loader", // creates style nodes from JS strings
                "css-loader", // translates CSS into CommonJS
                "sass-loader" // compiles Sass to CSS
            ]
        }]
    }
};

开发过程中，sourceMap功能非常有用，因为它可以帮助程序员快速定位有问题的代码。

添加sourceMap功能的实现：

module.exports = {
    ...
    devtool: "source-map", // any "source-map"-like devtool is possible
    module: {
        rules: [{
            test: /\.scss$/,
            use: [{
                loader: "style-loader"
            }, {
                loader: "css-loader", options: {
                    sourceMap: true
                }
            }, {
                loader: "sass-loader", options: {
                    sourceMap: true
                }
            }]
        }]
    }
};

3、生产阶段的建议。

代码开发、调试完成后，进入生产模式(production)。

此时，为了提升性能，需要对样式文件进行压缩。

安装插件

mini-css-extract-plugin

配置

const MiniCssExtractPlugin = require("mini-css-extract-plugin");

module.exports = {
    ...
    module: {
        rules: [{
            test: /\.scss$/,
            use: [
                // fallback to style-loader in development
                process.env.NODE_ENV !== 'production' ? 'style-loader' : MiniCssExtractPlugin.loader,
                "css-loader",
                "sass-loader"
            ]
        }]
    },
    plugins: [
        new MiniCssExtractPlugin({
            // Options similar to the same options in webpackOptions.output
            // both options are optional
            filename: "[name].css",
            chunkFilename: "[id].css"
        })
    ]
};
