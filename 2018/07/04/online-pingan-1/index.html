登录 api | 小程序作业专用

作者： Lian

日期： 2018年07月04日 16:07

小程序作业，实现登录接口，调试网络请求专用。

此接口已部署到云端，访问入口： https://test.liangyl.com

1 登录，获取token

1.1 网关：/user/login post

访问格式： ip + 端口 + 网关
如： https://test.liangyl.com/user/login

1.2 请求参数：

{
username:’admin’,
password:’123’
}

1.3 返回

// 正确

{
err_code: 0,
token: ‘xxx’
}

// 密码错误

{
err_code:1,
token: null
}

2 获取用户信息

2.1 网关： /user/info post

2.2 请求参数：

{
token: ‘xxx’
}

2.3 返回：

// 正确

{
err_code: 0,
username: ‘admin’,
age: 20
}

// 错误

{
err_code:1,
username:null
age:null
}

3 作业说明

这个作业挺简单，却又不简单。登录功能其实非常考核程序员的功力，它反映了程序员是否考虑周全。

1、登录之前，检测是否用户是否登录，进行合理跳转

2、输入参数，前端正则过滤，减少服务器请求次数，降低XSS风险

3、小程序原本的ajax请求采用异步嵌套方式，多层请求时，导致代码的可读性很丑

4、ajax请求的公共参数提取，异常拦截，都可以优化

5、根据用户输入，请求应答，给用户提供清晰，友好的提示。

6、登录成功，用户信息的缓存处理，与合理的页面跳转。
