1 知识点

1.1 什么是JavaScript？

一个前端web页面 = HTML(内容) + css(样式) + JavaScript(行为)

JavaScript用于描述网页的逻辑操作，交互行为。

1.2 JavaScript 与 html

link 标签，用于链接外部的js文件。通过这种方式，在HTML中添加JavaScript功能。

如：

<link rel="stylesheet" type="text/css" href="./style.css" />

2 实践

2.1 创建工程目录结构

  demo1 
  |- index.html
+ |- index.js

2.2 修改文件 index.html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>欢迎 登录</title>
    <link rel="stylesheet" type="text/css" href="./style.css" />
  </head>
  <body>

    <h1>登录(Login)</h1>

  </body>
</html>
