过程化编程

面向对象编程
设计类，将数据与方法合并起来，准确表示程序需要处理的东西。

泛型编程

结构
假如要存储运动员的信息，包括：姓名，年龄，工资，进球数等，希望有一种数据格式可以将所有数据存储在同一单元。
数组可以吗？数组只能存储相同格式的数据，所以不满足。
结构可以满足要求，因为同一个结构可以存储多种类型的数据。

#include <iostream>
struct inflatable
{
    char name[20];
    float volume;
    double price;
};
int main(){
    inflatable guest =
    {
        "Li ming",
        1.88,
        29.99
    };
    cout<< guest.name;
    return 0;
}

color paper = blue;

计算机在存储数据时，需要考虑3个基本问题：
1、信息存储在何处
2、存储的值是多少
3、存储的信息是什么类型

策略1，简单变量，声明语句定义了数据类型和标识符，程序为值分配内存，并在内部跟踪该内存位置。
策略2，定义指针，指针是一个变量，存储值的地址。*称为解除引用运算符，应用于指针，可以得到该地址存储的值。

int cups = 10;
// 声明指针
int * p_cups;
// 初始化
p_cups = &cups;

cout<< &cups;

分配内存与删除内存
指针真正的用武之地在于，分配未命名的内存以存储数据。

int * pt = new int;
*pt = 100;

delete pt;
