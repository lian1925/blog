CSS | 知识淬炼

作者： Lian

日期： 2018年02月07日 14:02

CSS

英文 Cascading Style Sheets，层叠样式表。
网页开发中，一个最重要的原则就是，内容与样式分离原则。
其中，HTML标签设计的初衷是用于定义文档内容，而CSS的出现，则是为了解决样式定义的问题。

CSS规则有两大组成部分：
1、选择器
2、一条或多条声明语句，每条声明由一个属性和值组成。
语法定义如下：

selector{
  declaration1;
  declaration2;
  ...
  declarationN
}

示例：

h1 {color:red; font-size:14px;}

这行代码的作用是，将 h1 元素内的文字颜色定义为红色，同时将字体大小设置为 14 像素。

选择器

要想使用CSS将样式应用于特定的HTML元素，需要想办法找到这个元素。在CSS中，执行这一任务的规则称为选择器。

元素选择器

元素选择器是最常见的选择器，通常将HTML文档中的元素直接当成css选择器使用。
示例：

html {color:black;}
h1 {color:blue;}
h2 {color:silver;}

类选择器

类选择器允许以一种独立于文档元素的方式来指定样式。为了将类选择器的样式与元素关联，必须将 class 指定为一个适当的值。请看下面的 HTML 代码：

<h1 class="top">
This heading is very important.
</h1>

然后，为名为top的class定义样式规则

.top {color:red;}

ID选择器

与类选择器相似，它也是独立于文档元素，不同的地方在于id选择器是唯一的。

<h1 id="top">
This heading is very important.
</h1>

然后，为名为top的class定义样式规则。它以#开头。

#top {color:red;}

注意：class 和id是区分大小写的。

#intro {font-weight:bold;}

<p id="Intro">This is a paragraph of introduction.</p>

在代码中，字体不会有粗体样式，因为大小写不一致。

后代选择器

这是按照上下文环境选择元素的一种方式，选择某元素的后代(嵌套)的元素。

<h1>This is a <em>important</em> heading</h1>
<p>This is a <em>important</em> paragraph.</p>

如果您希望只对 h1 元素中的 em 元素应用样式

h1 em {color:red;}

子元素选择器

如果你不希望选择任意的后代元素，而是希望缩小范围，只选择某个元素的子元素，请使用子元素选择器（Child selector）。

<h1>This is <em>really <strong>very</strong></em> important.</h1>

子元素选择器，以>连接父元素与子元素。

h1 > strong {color:red;}

相邻兄弟选择器

相邻兄弟选择器（Adjacent sibling selector），指紧接在另一元素后的元素，且二者有相同父元素。

<h1>This is a heading.</h1>
<p>This is paragraph.</p>
<p>This is paragraph.</p>

相邻选择器，以+相连

h1 + p {margin-top:50px;}

属性选择器

根据元素的属性及属性值来选择元素。
1、如果希望选择有某个属性的元素，而不论属性值是什么
示例：

a[href] {color:red;}

只对有 href 属性的锚（a 元素）应用样式：
2、根据具体属性值选择
除了选择拥有某些属性的元素，还可以进一步缩小选择范围，只选择有特定属性值的元素。

a[href="http://www.w3school.com.cn/about_us.asp"]
{
color: red;
}

将指向 Web 服务器上某个指定文档的超链接变成红色

伪类

用于向某些选择器添加特殊的效果。
1、锚伪类

a:link {color: #FF0000}        /* 未访问的链接 */
a:visited {color: #00FF00}    /* 已访问的链接 */
a:hover {color: #FF00FF}    /* 鼠标移动到链接上 */
a:active {color: #0000FF}    /* 选定的链接 */

2、:first-child 伪类
用于选择元素的第一个子元素。
示例：

<html>
<head>
<style type="text/css">
p:first-child {
  color: red;
  }
</style>
</head>

<body>
<p>some text</p>
<p>some text</p>
</body>
</html>

伪元素

1、:before 伪元素
该伪元素可以在元素的内容前面插入新内容。
示例：

<html>
<head>
<style type="text/css">
h1:before {content:url(/i/w3school_logo_white.gif)}
</style>
</head>

<body>
<h1>This is a heading</h1>
<p>The :before pseudo-element inserts content before an element.</p>
</body>
</html>

2、:after 伪元素
该伪元素可以在元素的内容之后插入新内容。

<html>
<head>
<style type="text/css">
h1:after {content:url(/i/w3school_logo_white.gif)}
</style>
</head>

<body>
<h1>This is a heading</h1>
<p>The :before pseudo-element inserts content before an element.</p>
</body>
</html>

盒模型

英文，box model，它的本质是一个盒子，用于封装HTML元素，由如下部分组成：
1、margin，外边距
2、border，边框
3、padding，内边距
4、content，盒子的内容

宽度
默认情况下，width值代表盒子的内容宽度，盒子的总宽度，还需要添加内边距，边框和外边距。
如：

div {
    width: 300px;
    border: 25px solid green;
    padding: 25px;
    margin: 25px;
}

例子中，盒子内容宽度为300px，盒子总宽度为300+252+252+25*2=450px

盒子模型中高度的概念与宽度相似。

box-sizing属性
CSS3，引入了一个新属性box-sizing，用于重新定义盒子模型的属性。
1、content-box，默认值，仅仅将内容放在盒子中，内边距，边框和外边距均在盒子外面。故盒子宽度=内容宽度。
2、border-box，将盒子的边框与内边距放入盒子中，外边距在外面。故盒子宽度=内容宽度+边框宽度+内边距宽度。
示例：

div
{
box-sizing:border-box;
-moz-box-sizing:border-box; /* Firefox */
}

border

边框，规定一个元素边框的样式和颜色。
1、border-style
规定边框的样式

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>border-style</title>
<style>
p.none {border-style:none;}
p.dotted {border-style:dotted;}
p.dashed {border-style:dashed;}
p.solid {border-style:solid;}
p.double {border-style:double;}
p.groove {border-style:groove;}
p.ridge {border-style:ridge;}
p.inset {border-style:inset;}
p.outset {border-style:outset;}
p.hidden {border-style:hidden;}
</style>
</head>

<body>
<p class="none">无边框。</p>
<p class="dotted">虚线边框。</p>
<p class="dashed">虚线边框。</p>
<p class="solid">实线边框。</p>
<p class="double">双边框。</p>
<p class="groove"> 凹槽边框。</p>
<p class="ridge">垄状边框。</p>
<p class="inset">嵌入边框。</p>
<p class="outset">外凸边框。</p>
<p class="hidden">隐藏边框。</p>
</body>

</html>

另外，可以单独指定上下左右边框的属性

p
{
    border-top-style:dotted;
    border-right-style:solid;
    border-bottom-style:dotted;
    border-left-style:solid;
}

2、border-width
规定边框的宽度。

p.one
{
    border-style:solid;
    border-width:5px;
}

3、border-color
规定边框颜色

p.one
{
    border-style:solid;
    border-color:red;
}

4、border
简写属性，可以将上述的属性合并。

p{
border: 5px solid red;
}

margin

规定外边距样式，主要是宽度信息。
程序员最常用的是它的简写属性。margin属性后面可以设置4个值，依次为上，右，下，左。

p{
  margin:10px 20px 30px 40px;
}

padding

定义元素边框与元素内容之间的空间。
用法与margin相似。

文本

文本是网页中非常重要的内容。我们来看看文本相关的样式规则。
1、color
规定文本颜色。

h1{
  color:#ff0000;
}

2、text-align
文本对齐方式，包括：左，右，居中，两端对齐(left right center justify)

p{
  text-align: center;
}

3、text-decoration
文本修饰符，在这里用于规定下划线的样式，包括：上划线，中划线，下划线，无划线(overline line-through underline none)

a{
  text-decoration: none;
}

4、text-transform
文本转换，用于大小写的转换，包括：全大写，全小写，首字母大写(uppercase lowercase capitalize)

span{
  text-transform: capitalize;
}

5、text-indent
文本缩进，一般用于段落中的首行缩进。

p{
  text-indent: 50px;
}

字体

用于规定字体，加粗，大小，文字样式。
1、font-familiy
规定字体种类，建议设置多个字体名称作为后备机制，如果浏览器不支持第一种字体，将依次尝试后面的字体。

body{
  font-family: sans-serif,"宋体";
}

2、font-style
字体样式，包括：正常，斜体(normal italic)
其实，还有第三种oblique，代表倾斜字体，与斜体重合，且浏览器支持度不好，故不推荐。

body{
  font-style: normal;
}

3、font-size
字体大小，通常有两种策略：
a)绝对大小，用px单位
b)相对大小，为了允许用户在浏览器改变字体大小，许多开发者使用em作为单位。1em与当前字体大小相等，浏览器默认文字大小为16px.

p{
  font-size: 16px;
}

链接

链接有4种状态：
a:link，正常，未访问过的链接。
a:visited，用户已访问过的链接。
a:hover，当鼠标悬停在链接上。
a:active，链接被点击的时刻。

a:hover{
  color:#ff0000;
}

背景

定义HTML元素的背景。
1、background-color
背景颜色

body{
  background-color: red;
}

2、background-image
背景图像

body{
  background-image: url('lian.jpg');
}

3、background-repeat
平铺，包括：水平平铺，垂直平铺，平铺，不平铺(repeat-x repeat-y repeat no-repeat)

body{
  background-repeat: repeat;
}

4、background-position
图片位置

body{
  background-position: center;
}

5、background
简写属性，合并相关属性

body{
  background: url('lian.jpg') no-repeat center;
}

display

显示属性，包括：可见与不可见，内联与块元素

p{
  display: none;
}

position

规定元素的定位类型，包括：
1、static，默认值，不会受到top,bottom,left,right影响。
2、relative，相对于其所在的正常位置。相对定位元素经常被用来作为绝对定位元素的容器块。

h1{
  position: relative;
  left: 20px;
}

3、absolute，相对于最近的已定位父元素。若是没有定位父元素，则相对于HTML

h1{
  position: absolute;
  top:10px;
  left: 10px;
}

4、fixed，相对于浏览器窗口的位置。

h1{
  position: fixed;
  top:10px;
  left: 10px;
}
