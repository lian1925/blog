移动端适配

策略，检测客户端是否是手机设备，若是，在body标签加入mobile类。
接下来，在body.mobile 的子元素中，对移动端适配样式调整即可。

第一步：在body的第一行加入如下脚本。

  <script>
      if (/mobile/i.test(navigator.userAgent) || /android/i.test(navigator.userAgent)) 
        document.body.classList.add('mobile');
  </script>

打开浏览器测试效果，当客户端为pc设备时，body无变化；当客户端为移动端时，body标签追加了mobile类。

第二步，初始化常用样式。

/* 样式复位 */

    /* // 边距 */
    margin: 0;
    padding: 0;
    /* // 文字 */
    list-style-type: none;
    text-align: left;
    text-decoration: none;
    font-weight: normal;
    font-style: normal;

    /* // 边框 */

    border: none;
};
