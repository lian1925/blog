JavaScript

JavaScript(缩写：JS)，一门轻量级的动态编程语言。当你需要开发web网站时，它可以帮助你实现丰富的交互功能。

任何一门技术，都是为了解决当时的时代背景下的特定需求而诞生，JavaScript当然也不例外。
JavaScript诞生于1995年，那是大部分用户还在使用电话拨号上网的年代，带宽很低，网速很慢。当时，人们上网提交表单，验证输入数据是否齐全，这些数据是否合法，这些都需要发送到服务器端去验证。每一次与服务器的数据交换都是对人们耐心的煎熬，毕竟拨号上网速度非常慢。
为了解决这个问题，美国的Netscape公司开发了一门编程语言，可以在用户的浏览器运行，帮助用户处理简单的表单数据，而不用每次都提交服务器，极大地改善了用户体验。这门可以在浏览器运行的编程语言就是JavaScript。
