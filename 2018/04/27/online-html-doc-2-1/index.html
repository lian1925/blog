HTML 元素 2.1 | HTML

作者： Lian

日期： 2018年04月27日 17:04

html元素(elements)

html文档由元素组合而成。什么是html元素？我们来看个示例

<p>My cat is very grumpy</p>

我们的HTML元素有3个组成部分：开始标签，结束标签，内容。
1、开始标签：以<开头，以>结束，标签中的第一个单词，是标签的名称，描述它的功能。接下来的其他单词，是标签的属性，用于描述标签的行为。
2、结束标签：以</开头，以>结束。大多数标签是成对存在的，良好的编程习惯，是不能忘了写结束标签。如：省略

元素的结束标签，就是很多前端程序员容易犯的错误。
3、内容：包含在开始标签和结束标签中的内容，很多时候是文本。
示例：

<p class="main">My cat </p>

嵌套元素

你可以把元素放在其他元素之中，我们称之为嵌套。我们一般称嵌套在外面的元素称为父元素，里面的元素称为子元素。注意，子元素的开始标签和结束标签都被包含在父元素里头。
我们来看个例子

<p>My cat is <strong>very</strong> grumpy.</p>

在这个例子中，p元素嵌套了strong元素，表示对very这个单词进行强调。

HTML元素有2两种重要的分类：
1、块级元素：该元素在页面中以块状方式呈现，即相对于前面的内容，它会另起一行，其后的内容也会被挤到下一行展现。常见的块级元素有：div,p
2、内联元素：通常出现在块级元素内部（嵌套），并包裹文档内容的小部分。它一个显著特性就是元素不换行。常见的内联元素有：a，span

<p>My cat is <span>very</span> grumpy.</p>
