promise 13 | ES6

作者： Lian

日期： 2018年04月25日 10:04

promise是es6提供的一种新的语法，用于解决异步问题。

promise可以理解为一种状态机，内部提供了3种状态。
1、Pending，表示未处理
2、Fulfilled，表示已成功
3、Rejected，表示失败

注意的是，状态间的跳转不可回退。

const fetchingPosts = new Promise((res, rej) => {
  $.get("/posts")
    .done(posts => res(posts))
    .fail(err => rej(err));
});

fetchingPosts
  .then(posts => console.log(posts))
  .catch(err => console.log(err));
