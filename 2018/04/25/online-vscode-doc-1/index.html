起步 1 | VS Code

作者： Lian

日期： 2018年04月25日 13:04

程序员开发软件项目，首先得选择一款好用的代码编辑器。目前，前端开发者常用的编辑器有：VScode，sublime，atom

这几款工具我都使用过，觉得最好用的还是VScode，界面好看，启动快，功能强大，简直爱不释手。

1 安装

1、打开浏览器，进入VSCode的官网：https://code.visualstudio.com/

2、根据自己的操作系统，选择合适版本下载

3、双击软件，进行安装。过程非常简单，相信每个同学都能完成。

2 汉化

F1 -> 语言设置 -> 修改为en-US -> 保存重启 VSCode

{
    // 定义 VSCode 的显示语言。
    // 请参阅 https://go.microsoft.com/fwlink/?LinkId=761051，了解支持的语言列表。
    // 更改此值需要重启 VSCode。
    "locale":"zh-cn"
}

3 配置

1、调整字体大小
code => 首选项 =》 设置，在用户设置中添加一行：

"editor.fontSize": 14

2、调整 tab 大小
code => 首选项 =》 设置，在用户设置中添加一行：

"editor.tabSize": 2,

4 常见快捷键

1、快速打开文件
Mac ⌘P (Windows, Linux Ctrl+P)

2、查找
Mac ⌘F (Windows, Linux Ctrl+F)

3、安装插件
Mac ⇧⌘P (Windows, Linux Ctrl+Shift+P)
