伪类选择器 4 | CSS

作者： Lian

日期： 2018年04月24日 19:04

属性选择器

根据元素的属性及属性值来选择元素。
1、如果希望选择有某个属性的元素，而不论属性值是什么
示例：

a[href] {color:red;}

只对有 href 属性的锚（a 元素）应用样式：
2、根据具体属性值选择
除了选择拥有某些属性的元素，还可以进一步缩小选择范围，只选择有特定属性值的元素。

a[href="http://www.w3school.com.cn/about_us.asp"]
{
color: red;
}

将指向 Web 服务器上某个指定文档的超链接变成红色

伪类

用于向某些选择器添加特殊的效果。
1、锚伪类

a:link {color: #FF0000}        /* 未访问的链接 */
a:visited {color: #00FF00}    /* 已访问的链接 */
a:hover {color: #FF00FF}    /* 鼠标移动到链接上 */
a:active {color: #0000FF}    /* 选定的链接 */

2、:first-child 伪类
用于选择元素的第一个子元素。
示例：

<html>
<head>
<style type="text/css">
p:first-child {
  color: red;
  }
</style>
</head>

<body>
<p>some text</p>
<p>some text</p>
</body>
</html>

伪元素

1、:before 伪元素
该伪元素可以在元素的内容前面插入新内容。
示例：

<html>
<head>
<style type="text/css">
h1:before {content:url(/i/w3school_logo_white.gif)}
</style>
</head>

<body>
<h1>This is a heading</h1>
<p>The :before pseudo-element inserts content before an element.</p>
</body>
</html>

2、:after 伪元素
该伪元素可以在元素的内容之后插入新内容。

<html>
<head>
<style type="text/css">
h1:after {content:url(/i/w3school_logo_white.gif)}
</style>
</head>

<body>
<h1>This is a heading</h1>
<p>The :before pseudo-element inserts content before an element.</p>
</body>
</html>
