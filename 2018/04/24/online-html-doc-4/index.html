head元素

head元素包含一些页面的元数据，用于定义网页的相关属性。这些内容不会在浏览器中显示。
示例：

<head>
  <meta charset="utf-8">
  <title>My test page</title>
</head>

接下来，我们来看看head元素常见的子元素。

1、title元素，用于为网页添加标题。

<head>
  <meta charset="utf-8">
  <!-- 此处添加标题 -->
  <title>My test page</title>
</head>

2、meta元素，用于定义网页属性的元数据。元数据的官方定义有点拗口，描述数据的数据。它含有页面的相关信息，包括样式、脚本及数据，能帮助一些软件 (如搜索引擎， 浏览器等等）更好地运用和渲染页面。
常见的开发技巧，把你网站的关键字放在keyword和description元数据中，可以帮助搜索引擎更好找到你的网站。

<meta charset="utf-8">
<meta name="description" content="淘宝网 - 亚洲较大的网上交易平台，提供各类服饰、美容、家居、数码、话费/点卡充值… 数亿优质商品，同时提供担保交易(先收货后付款)等安全交易保障服务，并由商家提供退货承诺、破损补寄等消费者保障服务，让你安心享受网上购物乐趣！" />
<meta name="keyword" content="淘宝,掏宝,网上购物,C2C,在线交易,交易市场,网上交易,交易市场,网上买,网上卖,购物网站,团购,网上贸易,安全购物,电子商务,放心买,供应,买卖信息,网店,一口价,拍卖,网上开店,网络购物,打折,免费开店,网购,频道,店铺" />

3、link元素，用于引入外部的文件。常见的有：引入外部的css文件。

<link rel="stylesheet" href="my-css-file.css">

此外，它还可以帮助你实现网站添加小图标的功能。

<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

4、script元素，用于嵌入JavaScript脚本。

<script src="my-js-file.js"></script>

5、style元素，包含文档的样式信息。
