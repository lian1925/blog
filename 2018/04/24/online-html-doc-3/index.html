注释

注释，拆分为”注”与”释”两个字，分别代表标注和解释。我们在编程时，往往会在某段代码附近，写上标注，用于解释这段代码的意图，这就是我们的注释。
我们来看看html中注释的语法规则：
注释以<!--开头，以-->结束，中间的文本则是我们的注释内容。注释的内容不会在浏览器中显示出来，但是我们可以在源代码中查看。

我们来演示一下

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <!-- 此处是我们的网页标题 -->
    <title>hello,world</title>
  </head>
  <body>

  </body>
</html>

和其他编程语言一样，如果没有注释，一个充满标签的原始HTML文档，很快就难以阅读。因此，强烈建议大家在编程时，培养注释的好习惯，保持清晰的编程思路。
