变量 1 | ES6

作者： Lian

日期： 2018年04月24日 20:04

在JavaScript中，声明变量有3种方式：var let const。其中var和let声明变量，而const声明常量。

1 var

1、var 是声明的变量，其作用域属于函数作用域。



function myFunction() {
  var myVar = "Nick";
  console.log(myVar); // "Nick" - myVar is accessible inside the function
}
console.log(myVar); // Throws a ReferenceError, myVar is not accessible outside the function.

2、var 声明的变量，具备声明提升的特性。

console.log(myVar) // undefined -- no error raised
var myVar = 2;

上面的代码，实际等效如下：

var myVar;
console.log(myVar) // undefined -- no error raised
myVar = 2;

2 let

let 是 es6 新引入的声明变量方法。它与var类似，两者声明的变量都支持重新赋值。

1、let声明的变量，其作用域属于块作用域。

function myFunction() {
  let myVar = "Nick";
  if (true) {
    let myVar = "John";
    console.log(myVar); // "John"
    // actually, myVar being block scoped, we just created a new variable myVar.
    // this variable is not accessible outside this block and totally independent
    // from the first myVar created !
  }
  console.log(myVar); // "Nick", see how the instructions in the if block DID NOT affect this value
}
console.log(myVar); // Throws a ReferenceError, myVar is not accessible outside the function.

2、let声明的变量，不具有声明提升的特性，这点与var明显不同。

console.log(myVar) // raises a ReferenceError !
let myVar = 2;

3 const

1、const也是es6新引入的方法，用于声明常量，不可重新赋值。

const myVar = "Nick";
myVar = "John" // raises an error, reassignment is not allowed

const myVar = "Nick";
const myVar = "John" // raises an error, re-declaration is not allowed

2、const 声明的变量，其作用域属于块作用域。


  if (true) {
    const myVar = "John";
    console.log(myVar); // "John"
    // actually, myVar being block scoped, we just created a new variable myVar.
    // this variable is not accessible outside this block and totally independent
    // from the first myVar created !
  }
  console.log(myVar); // "Nick", see how the instructions in the if block DID NOT affect this value

3、需要注意的是，const声明的常量，对于引用数据类型，可以重新赋值。
如对象：

const person = {
  name: 'Nick'
};
person.name = 'John' // this will work ! person variable is not completely reassigned, but mutated
console.log(person.name) // "John"
person = "Sandra" // raises an error, because reassignment is not allowed with const declared variables

如数组

const person = [];
person.push('John'); // this will work ! person variable is not completely reassigned, but mutated
console.log(person[0]) // "John"
person = ["Nick"] // raises an error, because reassignment is not allowed with const declared variables
