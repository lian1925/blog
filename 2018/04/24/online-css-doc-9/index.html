背景 9 | CSS

作者： Lian

日期： 2018年04月24日 20:04

背景

定义HTML元素的背景。
1、background-color
背景颜色

body{
  background-color: red;
}

2、background-image
背景图像

body{
  background-image: url('lian.jpg');
}

3、background-repeat
平铺，包括：水平平铺，垂直平铺，平铺，不平铺(repeat-x repeat-y repeat no-repeat)

body{
  background-repeat: repeat;
}

4、background-position
图片位置

body{
  background-position: center;
}

5、background
简写属性，合并相关属性

body{
  background: url('lian.jpg') no-repeat center;
}

display

显示属性，包括：可见与不可见，内联与块元素

p{
  display: none;
}

position

规定元素的定位类型，包括：
1、static，默认值，不会受到top,bottom,left,right影响。
2、relative，相对于其所在的正常位置。相对定位元素经常被用来作为绝对定位元素的容器块。

h1{
  position: relative;
  left: 20px;
}

3、absolute，相对于最近的已定位父元素。若是没有定位父元素，则相对于HTML

h1{
  position: absolute;
  top:10px;
  left: 10px;
}

4、fixed，相对于浏览器窗口的位置。

h1{
  position: fixed;
  top:10px;
  left: 10px;
}
