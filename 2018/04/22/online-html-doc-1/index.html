HTML

英文全称 Hypertext Markup Language， 超文本标记语言，用于创建web网页。

W3C

英文全称 World Wild Web consortium，一个国际委员会组织。它是专门为定义HTML和XHTML标准而成立的。

HTML哲学

我们应该把精力集中在文档的内容上。那些看上去很酷的页面只不过是全景中的一个亮点而已,有内涵的内容才会吸引越来越多的读者。

委员会在设计HTML标准之初，达成了这样一个约定：内容高于视觉表现。所以，我们在使用html时，需要按照他的设计方式来使用：表明文档的结构，以便浏览器能够正确显示它们的内容。

我们来看一下，最简单的HTML文档

<!-- 声明文档类型. 很久以前，早期的HTML(大约1991年2月)，文档类型声明类似于链接，规定了HTML页面必须遵从的标准 -->
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>hello,world</title>
  </head>
