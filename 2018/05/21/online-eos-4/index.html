EOS 词汇表

作者： Lian

日期： 2018年05月21日 14:05

Account 账户

An on-chain identifier【身份】 made up of native【本地】 and/or custom【自定义】 permissions【权限】 that are assigned【分配】 one or more keys【私钥】 or accounts.

Authority 授权

An abstract【抽象】 of permissions【权限】 that represent【呈现】 how permissions are organized【组织】 in reality【现实】 that are bound to【绑定】 an individual【个体】 or groups of individuals

Block 区块

A confirmable【确定】 unit【单元】 of the Blockchain【区块链】. Each block contains zero or more Transactions【交易】, as well as a cryptographic【密码学】 connection to all prior【上级】 blocks. When a block becomes “irreversibly【不可撤销】 confirmed” it’s because a supermajority【完整】 of Block Producers【生产者】 have agreed that the given Block contains correct Transactions. Once a Block is irreversibly confirmed, it becomes a permanent【永久】 part of the immutable【不可改变】 Blockchain.

DAC 分布式自治公司

Decentralized Autonomous Collective, or Decentralized Autonomous Corporation. Described in detail here (need link).

DAO 分布式自治组织

Decentralized Autonomous Organization.

Deferred【延期】 Transaction

A transaction that is created by a smart contract【智能合约】 for later【延迟】 execution at a specific future time. This future transaction can also create another future transaction to happen after itself. Thus, the deferred transaction opens the door to creating infinite【无限】 loops. The user authorizing the deferred transaction must have the bandwidth【带宽】 to execute that future transaction (this is evaluated when it’s scheduled), and the storage to store it until executed.

DLTs

Distributed【分布式】 Ledger【记账】 Technologies. A distributed ledger (also called a shared ledger, or referred to as distributed ledger technology) is a consensus【共识】 of replicated【复制】, shared, and synchronized digital data geographically【地理】 spread across multiple sites, countries, or institutions【机构】. https://en.wikipedia.org/wiki/Distributed_ledger

DPoS

Delegated【委派】 Proof【证明】 of Stake【股份】. Also, “Democracy【民主】 as Proof of Stake.” DPoS is one of a collection of consensus algorithms, i.e. methods by which block producers can agree (reach consensus) on which transactions and which blocks are “real” and should be confirmed and treated as irreversible【不可撤销】.

Key pair 密钥对

A public key【公钥】 and its corresponding【对应】 private key【私钥】

larimer 货币单位

1/10000 of an EOS (token) 0.0001 EOS

Master Password 主密码

The password used to to unlock【解锁】 (decrypt) a wallet file

Action 行为

A change【改变】 to the Blockchain. One or more actions make up【组成】 a Transaction.

Non-Producing Node 非生产节点

A full node running nodeos that is only watching【观察】 and verifying【验证】 for itself each block, and maintaining【维持】 its own local full copy【副本】 of the blockchain. A non-producing node that is in the “standby pool” can, through the process of being voted【投票】 in, become a producing node. A producing node, if voted out【投票出局】, will become a non-producing node. Most non-producing nodes are not in the “standby pool【备用池】”.

Oracle 预言机

“An oracle, in the context【上下文】 of blockchains and smart contracts【智能合约】, is an agent that finds and verifies real-world occurrences【事件】 and submits【提交】 this information to a blockchain to be used by smart contracts.” Source

peer-to-peer 点对点

peer-to-peer computing or networking is a distributed【分布】 application architecture that partitions【分割】 tasks or workloads between peers. Peers are equally privileged【特权】, equipotent participants in the application. They are said to form a peer-to-peer network of nodes. https://en.wikipedia.org/wiki/Peer-to-peer

Permission 权限

A weighted security mechanism【机制】 that determines whether or not a message is properly【合理】 authorized【授权】 by evaluating【评估】 its signature(s) authority

Private Key 私钥

A secret key used to sign transactions

Public Key 公钥

A publicly available key that is transmitted alongside a transaction

Scope 范围

Scope is a region【范围】 of data within a contract【合约】. Contracts can only write【写入】 to regions in their own contracts but they can read【读写】 from any contract’s regions. Proper scoping allows transactions to run in parallel【并行】 for the same contract because they do not write to the same regions【相同范围】. Scope is not conflated【合并】 with an account name, but contracts can use the same value for both for convenience.

Smart Contract 智能合约

A smart contract is a computer【计算机】 protocol【协议】 intended【试图】 to facilitate, verify【检验】, or enforce the negotiation【协商】 or performance of a contract.

Standby【备选】 Pool【池】

A set of about 100 full nodes【节点】 that have expressed【表达】 the desire【要求】 to be selected【选举】 as block producers, and are capable of doing so on demand. Whenever the chain needs to replace【替换】 an existing BP with a new one, the new one is drawn from the standby【备选】 pool.

Transaction【交易】

A complete all-or-nothing change【变化】 to the Blockchain. A combination【组合】 of one or more Messages. Usually, the execution【执行】 of a Smart Contract.

Wallet 钱包

An encrypted【加密】 file generated【生成】 and/or managed by a client【客户端】 (for example, cleos) that manages private【私钥】 keys and facilitates【帮助】 the signing of transactions in a secure【安全】 manner【方式】. Wallets may be in a locked or unlocked state【状态】.

Block Producer 区块生产者

The node【节点】 that is currently【当前】 taking its turn producing the “right now” block for the blockchain【区块链】. Or, a member【成员】 of the group of nodes who have been elected【选举】 to take such turns. Synonymous【同义词】 with ‘block producer’
