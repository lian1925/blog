Features【特性】 of the JavaScript Editor【编辑器】

作者： Lian

日期： 2018年05月26日 08:05

1、来源 monaco
微软公司在GitHub开源了一款很好用的编辑器，称为monaco。我们的pxt编辑器，正是基于monaco进行扩展开发。

功能
2、着色
对不同功能区块，分别进行着色，便于用户更好区分。

3、单词补全
单词补全，是指我们在编程时，只需要输入单词的前几个字母，编辑器会自动推测我们最可能想要输入的内容，帮助我们补全。
举个例子：如输入function，当我们输入前3个字母fun时，编辑器会帮忙补全为function。

4、参数提示
我们知道，函数是可以接收参数的。当我们调用一个函数时，编辑器会自动提示我们传入参数的个数，类型，相关解释。

5、快速提示
当我们阅读代码时，看到一个陌生的函数时，我们想知道该函数的作用和用法，应该怎么操作？
最常用的方法，是找到函数的定义处，了解函数是如何实现，进而知道它的作用。
这种方法的缺点是很花时间，开发成本高。
快速提示功能给我们提供另一种解决方案，遇到陌生函数时，我们只需要将鼠标停留在函数上方一会，函数的相关信息则自动出现。

6、括号匹配
编程时，我们会经常使用到括号。一般用在两种地方：
传参，我们在函数传参时，用普通括号括起来。
函数体，我们通常会将函数功能实现写在函数体内，此时，我们使用大括号将函数体内容包括起来。
我们的括号总是成对出现，有开始括号，就得伴随结束括号。由于编程时，经常遇到各种语句相互嵌套的情况，单单靠眼睛很难知道哪对括号是一起的。
有了编辑器的括号匹配功能，我们只需要将鼠标点在某个括号旁，则自动提示我们另外匹配的括号。

7、错误提示
编辑器提供了代码实时分析功能，当检测到代码出现错误时，会自动提醒用户。
举个例子：
当函数名称拼写错误，或传参个数不匹配，编辑器在错误处，用红色的波浪线标示出来。

Features of the JavaScript Editor

The JavaScript editor extends【延展】 the features and functionality【功能】 of the Monaco Editor, the editor that powers【引擎】 Visual Studio Code.

Here is a list of some of the features supported【支持】 by the Editor:

Editing【编辑】

The Editor contains【包含】 a built-in【内置】 JavaScript/Typescript language service that provides complete【完整】 code intelligence【智能】.

Coloring【着色】

The Editor automatically【自动】 colors each of function and its respective namespace【命名空间】 to match the colors of the respective block【块】 in the Block Editor.

This is the equivalent【等效】 of the following blocks:

Auto Suggest【建议】

Also known as IntelliSense, the editor supports【支持】 automatic word completion【补全】.

If the language service knows possible【可能】 completions, the IntelliSense suggestions will pop up as you type. You can always manually trigger it with Ctrl and Space.

You can get additional【额外】 information if you click the ‘i’ symbol in the current suggestion.

Auto completion【补全】

Pressing Enter to any of the functions suggested by IntelliSense will automatically insert【插入】 a snippet of the function declaration【声明】 prefilling default values for each of the function parameters【参数】.

The default【默认】 value is populated in one of the following ways:

  • The default value in a @param JSdoc in the function declaration【声明】.
  • Default values for primitive【原生】 types. number: 0, string: "", boolean: false
  • If a parameter is an Enum【枚举】, the first value of the Enum is inserted

Parameter【参数】 Hints

Parameter hints will pop up as you’re typing a function invocation.

Quick Info【信息】

Hovering over namespaces【命名空间】, functions and function parameters will show useful【有用】 information describing the purposes of the function, namespace, or parameter.

Bracket Matching【匹配】

Matching brackets are highlighted【高亮】 as soon as the cursor is near one of them.

Tip: You can jump to the matching bracket with Ctrl and Shift and \

Errors【错误】

The Editor’s language【语言】 service is constantly analyzing【分析】 your code for errors in the background.
Errors are surfaced to the user in one of two different ways:

  • Red squiggly or wavy lines appear【出现】 beneath your code that is incorrect
  • An Error bar snaps to the first error in the list

Zoom【缩放】

You can change the editor’s font size by zooming in or out the editor content【内容】.
Use Ctrl and + and Ctrl and - to zoom in/out.

Find【寻找】 and Replace【替换】

The Editor supports Find, as well as Find and Replace in order to search for a particular keyword, or search and replace a particular keyword.
You can get to the Find and Replace widget with Ctrl and f, or you can also get to it via the All Commands window (see below).

Context Menu【菜单】

Right clicking anywhere within the editor will bring up the context menu.
The context menu will show you quick actions that you can complete by clicking one of the buttons that appear in the context menu.

Here are some of the actions【行为】 currently supported:

  • Format Code: Quickly formats the document code
  • Show Commands: Opens up the All Commands view
  • Save: Saves the document
  • Run Simulator: Runs the simulator

All Commands【命令】

The All Commands view allows【允许】 the user to search through all the commands available in the editor.
To get to the all commands view press F1, or right click to bring up the context menu, then click Show Commands.

Language Support【支持】

The Monaco editor is also used for displaying other【其他】 languages, including JSON, Cpp, Text and Assembly files.
