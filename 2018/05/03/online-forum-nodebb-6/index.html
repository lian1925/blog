国际化 6 | 技术实现

作者： Lian

日期： 2018年05月03日 21:05

nodebb提供一个语言引擎，帮助你实现多国语言。

1 语言文件

为你的项目添加语言文件，如 languages/en 英文和 languages/ch 中文。

$ cd /path/to/my/plugin
$ mkdir -p languages/en
$ mkdir -p languages/ch

2 添加翻译词组

在languages/en/plugin.json

{
    "greeting": "Hello there! How are you?"
}

在languages/ch/plugin.json

{
    "greeting": "你好！最近可好？"
}

3 告诉nodebb语言文件的位置

找到plugin.json文件，添加一行

{
    ...
    "languages": "languages",
    "defaultLang": "es",
    ...
}

4 使用语言文件

1、在服务器端使用

var translator = require.main.require('./public/src/modules/translator');

translator.translate('[[myplugin:greeting]]', function(translated) {
    console.log('Translated string:', translated);
});

2、客户端

require(['translator'], function(translator) {
    translator.translate('[[myplugin:greeting]]', function(translated) {
        console.log('Translated string:', translated);
    });
});

3、模板文件

<p>[[myplugin:greeting]]</p>
