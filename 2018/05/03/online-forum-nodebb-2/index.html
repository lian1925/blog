安装 2 | 技术实现

作者： Lian

日期： 2018年05月03日 16:05

今天，我们来看看如何在Mac系统和Linux系统安装NodeBB。

1 Mac

1、安装 redis，用于存储数据

brew install redis

2、启动 redis

redis-server

3、下载 NodeBB 源码

git clone -b v1.9.x https://github.com/NodeBB/NodeBB.git

4、进入源码目录

cd NodeBB

5、执行安装脚本

./nodebb setup

注意：

  • 当提示数据库时，输入 redis
  • 数据库 lian, 密码 liannode

此处，需要等待一会。。。

6、启动论坛

./nodebb start

7、访问论坛

http://localhost:4567/

2 Ubuntu

安装 Nodejs

$ curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
$ sudo apt-get install -y nodejs

MongoDB

1、安装 mongoDB

$ sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv 2930ADAE8CAF5059EE73BB4B58712A2291FA4AD5
$ echo "deb [ arch=amd64,arm64 ] http://repo.mongodb.org/apt/ubuntu xenial/mongodb-org/3.6 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-3.6.list
$ sudo apt-get update
$ sudo apt-get install -y mongodb-org

2、检查MongoDB版本

$ mongod --version
db version v3.6

3、启动MongoDB

$ sudo systemctl start mongod
$ sudo systemctl status mongod

配置MongoDB

1、进入数据库

$ sudo systemctl start mongod
$ sudo systemctl status mongod

2、进入 admin 数据库

use admin

3、添加数据库管理员admin

> db.createUser( { user: "admin", pwd: "<Enter a secure password>", roles: [ { role: "readWriteAnyDatabase", db: "admin" }, { role: "userAdminAnyDatabase", db: "admin" } ] } )

4、添加新数据库 nodebb

> use nodebb

5、离开 MongoDB

 quit()

6、修改MongoDB配置文件 /etc/mongod.conf

security:
  authorization: enabled

7、重启与验证管理员账号

$ sudo systemctl restart mongod
$ mongo -u admin -p your_password --authenticationDatabase=admin

安装 NodeBB

1、安装 git，用于下载nodebb源码

sudo apt-get install -y git

2、下载 nodebb 源码

$ git clone -b v1.8.x https://github.com/NodeBB/NodeBB.git nodebb
$ cd nodebb

3、执行安装脚本

$ ./nodebb setup

4、启动nodebb

$ ./nodebb start

nginx

1、安装

$ sudo add-apt-repository ppa:nginx/stable
$ sudo apt-get update
$ sudo apt-get install -y nginx

2、检查版本

$ nginx -v
nginx version: nginx/1.12.2

3、启动nginx 

$ sudo systemctl start nginx
$ sudo systemctl status nginx

配置 nginx

1、在 /etc/nginx/sites-available目录下，添加一个配置文件，如：/etc/nginx/sites-available/forum.example.org

$ cd /etc/nginx/sites-available
$ sudo nano forum.example.com # config entered into file and saved

2、设置软链接，关联配置文件，至sites-enabled.

$ cd ../sites-enabled
$ sudo ln -s ../sites-available/forum.example.com

3、配置文件示例：

server {
    listen 80;

    server_name forum.example.com;

    location / {
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header Host $http_host;
        proxy_set_header X-NginX-Proxy true;

        proxy_pass http://127.0.0.1:4567;
        proxy_redirect off;

        # Socket.IO Support
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}

5、重启NGINX

sudo systemctl reload nginx
