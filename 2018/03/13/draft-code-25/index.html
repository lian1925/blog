1、如何实现 水平垂直 居中？

答：

分两种情况，

一种是高度与宽度已知，处理这种情况最直接的方式就是简单计算出上边距与左边距。

第二种，高度与宽度未知，这种情况比较复杂，解决思路有两种：
利用 transform 移动。

#container{
    position:relative;
}

#center{
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

利用flex布局

#container{
    display:flex;
    justify-content:center;
    align-items: center;
}

#center{

}

2、inline 、 block 和inline-block的区别？

答：

inline元素嵌入行元素之间，宽度由内容决定，无法通过width指定。

block，元素新起一行，可以设置宽度

inline-block，从外面看是inline，嵌入行中；从外看，是block，可以通过width设定宽度。

3、CSS中link和@import的区别？

答：

一，link是XHTML标签，除了加载css，还有其他功能，如加载图片和 js;
import 是css语法，只能加载css

二，link引用css，页面载入时同时加载操作；
import 则等页面完全载入后，才能加载。

4、JavaScript的严格模式是什么？

答：

使用严格模式时，在执行语句前加入

"use strict";

最大的用途有3种：
一、消除JavaScript语法的一些不合理，不严格之处，减少怪异行为；

二、消除代码运行的一些不安全之处；

三、提高编译器效率，增加运行速度

5、call和 apply 的区别？

答：

两者作用一致，都是把函数fun1绑定到obj，这时候obj具备了该方法，可以当成自己的方法来调用。如果方法中有 this,指向obj.

唯一区别是apply接受的是数组参数，call接受的是连续参数。

fun1.call(obj, arg1, arg2, ...);
fun1.apply(obj, [arg1, arg2, ...]);

(完)
