1、 == 与 === 有何不同？

答：

== 称为相等运算符，比较时，先将操作数转换成数值，再比较它们是否相等。

=== 称为严格相等运算符，比较时，直接判断他们是否相等。

比如：

1 == true // true
1 === true // false

2、如何清除浮动？

答：

大致有三种思路：

一、为父元素设置高。

二、在浮动元素后面添加元素,清除浮动。

clear:both;

三，使用伪元素选择器，这是推荐做法，bootstrap框架即使用了该做法。

.clear:after{
  content:"";
  display:block;
  clear: both;
}

3、请解释一下什么闭包？

答：

闭包，closure，指能够读取其他函数的内部变量的函数。

比如：

function f1(){
　　var n=0;
　　nAdd=function(){n+=1}
　　function f2(){
　　　　alert(n);
　　}
　　return f2;
}
var result=f1();
result(); // 999
nAdd();
result(); // 1000

闭包最大的用途有两个：
一、读取函数的内部变量；
二、让绑定的变量值始终保持在内存中。

缺点是滥用闭包，造成内存消耗过度，出现性能问题，甚至导致内存泄漏。

4、cookie，sessionStorage和localStorage的区别？

答：

cookie，存储数据量小,不超过4k，可以设置存储时间，到期删除；

sessionStorage临时存储数据，可以达到5m，离开当前页面数据删除。有意思的地方在于，它引入了 浏览器窗口 的概念，在同源的相同窗口，数据始终存在，即使刷新也不影响。只有在关闭页面时才会删除数据。

localStorage，可以永久存储。

另外，cookie的数据会自动的传递到服务器，服务器端也可以写cookie到客户端。
sessionStorage和localStorage不会自动把数据发给服务器，仅在本地保存。

5、em 与 rem 的区别？

答：

em 与 rem 都是计算相对单位。
不同之处在于：rem针对html根元素的font-size值，计算尺寸。
而 em 则相对父元素的font-size计算。
