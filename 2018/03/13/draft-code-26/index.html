1、什么是渐进增强和优雅降级？

答：

渐进增强，针对低版本浏览器构建页面，保证最基本的功能；然后针对高版本浏览器，改进效果和交互，提高用户体验

优雅降级，则相反。先提高较高的用户体验，再针对低版本浏览器做兼容设计。

2、href 和 src 的区别？

答：

src，source的简称，指向物体的来源。常用在img,script等处。

href，全称为 hypertext reference，超文本引用，常在 link 和 a 元素等处使用。

3、什么是jsonp?

答：

jsonp，是解决跨源通信的常用方法。其优点是简单，对老版本浏览器兼容性好。

基本策略是：向网页添加一个<script>元素，向服务器请求json数据，此时不受同源政策的影响。服务器正常应答后，将数据放在一个指定的回调函数。

简单实现如下：

function addScriptTag(src) {
var script = document.createElement(‘script’);
script.setAttribute(“type”,”text/javascript”);
script.src = src;
document.body.appendChild(script);
}

window.onload = function () {
addScriptTag(‘http://example.com/ip?callback=foo‘);
}

function foo(data) {
console.log(‘Your public IP address is: ‘ + data.ip);
};

上述代码执行后，数据通过foo可以获取。
