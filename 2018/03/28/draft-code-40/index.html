自我介绍

毕业后，校招进入了tplink公司，从事研发工作。

后来，跳槽去了深圳爱海购公司，当技术负责人，负责电商平台的搭建。

现在，他们公司找到了合适的负责人，我也一直希望能到大公司发展，所以过来这里面试。
