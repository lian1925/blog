code 1

摘自

https://github.com/samuel1112/v2er/blob/master/src/Utils/API.js

function _obj2uri(obj){
    if(!obj){
        return '';
    }
    return '?'+Object.keys(obj).map(function(k) {
        return encodeURIComponent(k) + "=" + encodeURIComponent(obj[k]);
    }).join('&');
}

理由：

api请求场景中，将object转化为url格式是常见的需求。使用Object.key().map和join的连用，是很优雅的写法。

code 3

摘自

https://github.com/samuel1112/v2er/blob/master/src/TopicDetail.js

function getTopicInfo(topics, id){
    let _res = {};
    topics.forEach(topic=>{
        if(topic.id === id){
            _res = topic;
            return false;
        }
    });
    return _res;
}

理由：

在一组数组对象中，寻找特定的值，是一种常见的需求。forEach与return false连用，是一种常规解法。

function topicType(state={type: 'latest', name: 'Latest'}, action){
    switch (action.type){
        case SET_TOPIC_VIEW:
            return action.topic;
        default:
            return state;
    }
}

理由：

es6的新语法中，允许定义函数时，声明传参的默认值。在redux的reducer中，使用该特性是非常合适的。
