code 1

https://github.com/bailicangdu/vue2-elm/blob/master/src/store/action.js

async getUserInfo({
    commit,
    state
}) {
    let res = await getUser();
    commit(GET_USERINFO, res)
}

理由：

api请求时，往往接触到异步操作，比较优雅的做法是将异步转同步。

code 3

摘自

https://github.com/bailicangdu/vue2-elm/blob/master/src/router/router.js

const home = r => require.ensure([], () => r(require('../page/home/home')), 'home')
const city = r => require.ensure([], () => r(require('../page/city/city')), 'city')

理由：

这里用到了require.ensure，它是webpack提供的代码分割功能。

它的作用是把js模块给独立导出一个.js文件的，然后使用这个模块的时候，webpack会构造script dom元素，由浏览器发起异步请求这个js文件。

code 5

摘自

https://github.com/bailicangdu/vue2-elm/blob/master/src/config/fetch.js

if (type == 'GET') {
    let dataStr = ''; //数据拼接字符串
    Object.keys(data).forEach(key => {
        dataStr += key + '=' + data[key] + '&';
    })
    if (dataStr !== '') {
        dataStr = dataStr.substr(0, dataStr.lastIndexOf('&'));
        url = url + '?' + dataStr;
    }
}

理由：

get请求时，往往需要将数据与url拼接，这是一种传统的处理方式。相比而言，上文提到的处理方式，更为简洁优雅。
